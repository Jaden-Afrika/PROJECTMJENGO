# PROJECT MJENGO

A real-time construction ledger and material tracking workspace for site managers in Kenya.


## ABOUT

**Project Mjengo** is built to take the chaos out of managing construction logistics and finances. It is a secure, cloud-based construction ledger designed specifically for developers and site managers in Kenya who need an unambiguous, real-time interface to track building materials, labor costs, and site budgets.

Users can instantly log asset deliveries, monitor material spending, track vendor bills, and evaluate fluctuating costs right on the spot. The primary objective is to drive complete transparency into building processes, eliminating financial blind spots and giving project heads a zero-headache, real-time cash flow overview.


## WEBSITE DESIGN

When a site manager logs into the application, they are greeted by a high-performance central control room layout. The workspace features immediate, live-updating cards displaying the overall project budget, total fundi/labor payouts, and a continuously updating remaining balance calculated instantly in Kenyan Shillings (KES).

The interface uses a premium, high-contrast, structured aesthetic built by a **Software Engineer** who knows how to optimize field workflows. Layout actions allow managers to seamlessly drop down logs, input material quantities, or toggle between an immersive **Project Selection Dashboard** and individual live site data streams. The design utilizes deep slate backdrops, sharp content containers, and subtle structural watermarks to ensure maximum legibility and zero visual clutter under any field lighting.

## TECH STACK

This application is engineered for zero local footprint, fast execution, and modern component lifecycle management:

* React.js (JSX) – Component-driven frontend layout architecture and fluid UI rendering.
* Tailwind CSS – High-performance utility classes, frosted-glass overlay layouts, and immediate mobile-responsive grid formatting.
*Firebase (Firestore & Auth)– Handles real-time ledger stream syncing and secure Google account authorization.

---

## SERVICES USED

The system relies on cloud infrastructure for absolute durability and data synchronization:

* Firebase Authentication** – Powers secure, zero-footprint Google account login pipelines.
* Cloud Firestore** – Acts as the live data pipeline, executing immediate transaction calculations and keeping multi-user data synced.
